# Boilerplate

nodejs (express - twig), webpack 5, mongodb

Minimum node version : 14.15

nvm use 14.15

## Installation

Clone this repo and npm install.

```bash
npm i
```

## Usage

### Development server

```bash
npm run start
npm run webpack
```

You can view the development server at `localhost:8080`.

### Production build

```bash
npm run build
```
